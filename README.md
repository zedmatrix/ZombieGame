# ZombieGame
Humans vs Zombies C++23 SDL/OpenGL Game
#
* Learning the meson.build system
* Fundamentals of C++23 Programming
* SDL and OpenGL Programming
* Simple Graphic Image Creation

# Installation 
not tested, but build instructions:
```
mkdir -v build
cd build
meson setup ..
ninja
./zombie-game
```
